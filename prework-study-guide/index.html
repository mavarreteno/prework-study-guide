Prework Study Guide

HTML

  • The head is the hub for information about the webpage which is mostly invisible to the user
  • The body is the meat and potatoes of what the user interacts with and sees
  • Each element is preceded and followed by a tag "<...>"
  • The closing tag has a back slash after the opening bracket "< /...>"

CSS

  • inline CSS are HTML tags which style a particular HTML element only
  • internal style sheet is a style element inside the head element
  • most common is external style sheet which is a seperate file which is then linked
  • to link you add a "link" tag to the head
  • margin determines space around the element
  • padding is how much space there is around the element
  • place a period before as a class selector
  • border is the colored outline around the element
  • you cam modify box-shadow by its x offset and then y offset and then color
  • HTML Color Codes Website can show you the hexadecimal code of a color

Git

  • git status: checks which branch we are on
  • git checkout -b "branch name": creates a new branch and switches to it
  • from feature branch, git add -A (case sensitive): add ALL changes to staging area of git
  • git commit -m "blah": commit changes and add a little message
  • git push origin main: pull changes from origin remote, master branch and merge them to local
  • git push origin: push up our local changes
